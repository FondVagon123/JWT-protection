echo "# Flask JWT API

Цей проект демонструє простий Flask-додаток з використанням JWT для захисту API.

## Опис

Цей додаток реалізує REST API для керування товарами. Для доступу до деяких ендпоінтів необхідно авторизуватись через JWT (JSON Web Token). Додаток використовує Flask, Flask-SQLAlchemy для роботи з базою даних, а також Flask-JWT-Extended для аутентифікації.

## Як запустити проект

### Крок 1: Встановлення залежностей
Для початку потрібно встановити всі необхідні бібліотеки:
\`\`\`bash
pip install -r requirements.txt
\`\`\`

### Крок 2: Ініціалізація бази даних
Після встановлення залежностей необхідно створити міграції для бази даних:
\`\`\`bash
flask db init
flask db migrate -m \"Initial migration\"
flask db upgrade
\`\`\`

### Крок 3: Запуск додатку
Запустіть сервер Flask:
\`\`\`bash
python app.py
\`\`\`

Тепер додаток буде працювати на \`http://localhost:5000\`.

## Ендпоінти

### 1. **/login**  
   **Метод**: \`POST\`  
   **Опис**: Отримання JWT токена (логін).  
   **Приклад тіла запиту**:
   \`\`\`json
   {
       \"username\": \"admin\",
       \"password\": \"password\"
   }
   \`\`\`

### 2. **/logout**  
   **Метод**: \`POST\`  
   **Опис**: Вихід із системи (відкликання токена).

### 3. **/item**  
   **Метод**: \`GET\`, \`POST\`, \`DELETE\`  
   **Опис**: CRUD операції над товарами (створення, отримання, видалення).
